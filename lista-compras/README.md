### Manual de usuario

## USANDO LA APP

1. Presionar en "Agregar producto"

2. Escribir el nombre del producto que se agregará en la lista y luego presionar en "Agregar"

3. Marcar los productos ya comprados en la cajita de checklist

Para borrar un producto de la lista, mantener presionado sobre su nombre

## EJECUCIÓN DE LA APP EN UN ENTORNO LOCAL

1. Clonar el repositorio y ubicarse en el proyecto:
   git clone https://github.com/rocioalbarracin08/DDS-RocioA.git
   cd DDS-RocioA/lista-compras

2. Instalar dependencias
   bun install

3. Ejecutar la aplicación
   bun start

4. Instalar Expo Go en tu celular (PlayStore o AppStore)
   Abrir la aplicación

5. Abrir el Scanner dentro de la app y usarlo para scanear el QR generado por bun start

6. Cortar la ejecución
   Ctrl + "c"

Presionar "R" en caso de querer recargar la app
